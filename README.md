## Sobre o projeto
É uma aplicação em NodeJS que se alimenta da <a href="https://digimon-api.herokuapp.com">Digimon API</a> e retorna os Digimons de acordo com o nome ou level deles.

	
## Tecnologias
Atualmente rodando com:

- [x] NodeJS
- [x] Express
- [x] CommonJS
- [x] Nodemon
- [x] Body-Parser
